# 批处理
```html
{%for q in qs%}
<div onclick="batch_handle(type)">
<input class="batch-checkbox" type="checkbox" value={{q.id}}>
...
</div>
{%endfor%}

<script>
    function batch_handle(type){
        var q_box = document.getElementsByClassName('batch-checkbox');
        var q_ids =[];
        for (q in q_box){
            if (q_box[q].checked)
                q_ids.push(q_box[q].value);
        }
        console.log(q_ids);
    }
</script>
```
